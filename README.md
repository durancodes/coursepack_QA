# 📚 CoursePack QA - RAG System

An AI-powered question-answering system that allows students to upload PDFs and ask questions.

## 🚀 Features
- Multi-document PDF upload
- Hybrid Search (FAISS + BM25)
- Reranking for accuracy
- Student-friendly answers
- Chat-based UI using Streamlit

## 🧠 Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- Groq API (LLM)

## 📸 Demo
Upload → Ask → Get answers from your documents

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
